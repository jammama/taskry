## 📖 기능 개요

### 목적
Tasks의 디자인을 초기 디자인(`design.png`)과 동일하게 변경합니다. 태그의 배경 색상을 제거하고, 태그 위치를 완료버튼 왼쪽으로 옮기며, Task의 높이를 줄이고, 아이콘과 숫자 간격을 줄이고, XP 표시를 제거하고, 완료 버튼 크기를 줄여 UI를 더 깔끔하고 미니멀하게 만듭니다.

### 사용자 스토리
- **As a** 사용자
- **I want** Tasks가 깔끔하고 미니멀한 디자인
- **So that** UI가 복잡하지 않고 집중할 수 있다

---

## 🎯 기능 요구사항

### 핵심 기능
1. Tasks 디자인을 `design.png`와 동일하게 변경
2. 태그 배경 색상 완전 제거
3. 태그 위치를 Task 제목 오른쪽으로 이동
4. Task 높이를 디자인과 동일하게 줄이기
5. 아이콘과 숫자 간격 줄이기
6. XP 표시 완전 제거
7. 완료 버튼 크기 줄이기

### 상세 요구사항
- **태그 배경 색상 제거**: 
  - 태그의 배경색(`background: rgba(...)`) 완전 제거
  - 태그는 텍스트만 표시 (배경 없음)
- **태그 위치 변경**:
  - 현재: Task 제목 아래 또는 오른쪽에 표시
  - 변경: Task 제목 왼쪽에 표시 (숫자와 아이콘 사이 또는 아이콘 왼쪽)
- **태그 색상**:
  - 미묘한 회색 톤으로 표시 (강한 색상 제거)
  - `text-shadow`, `font-weight: bold` 등 강조 효과 제거
- **Task 높이 줄이기**:
  - 현재 Task 항목의 `padding` 줄이기
  - 태그가 제목 아래에 있지 않도록 하여 수직 공간 절약
  - Task 간 간격 줄이기
- **아이콘과 숫자 간격 줄이기**:
  - 숫자(`.index`)와 아이콘(`.icon`) 사이의 `margin` 줄이기
  - 더 컴팩트한 레이아웃
- **XP 표시 제거**:
  - XP 배지(`+{task.xp} XP`) 완전 제거
  - 태그 옆에 표시되던 XP 관련 요소 모두 제거
- **완료 버튼 크기 줄이기**:
  - 현재 완료 버튼(`.check-btn`)의 크기 줄이기
  - 디자인과 동일한 작은 크기로 조정

## 작업 완료 내역

### 변경 사항
- `src/lib/components/PlanningMode.svelte`: Tasks 디자인을 초기 디자인으로 복원 (태그 배경 제거, 태그 위치 변경, Task 높이 줄이기, 아이콘 간격 줄이기, XP 제거, 완료 버튼 크기 줄이기)

### 주요 수정 사항
1. **태그 배경 색상 완전 제거**
   - `.tag-focus`, `.tag-rhythm`, `.tag-catalyst` 개별 클래스 제거
   - 통합된 `.tag` 클래스로 변경
   - 배경색(`background`) 완전 제거
   - 태그는 텍스트만 표시 (배경 없음)

2. **태그 위치 변경 (제목 오른쪽으로 이동)**
   - 태그를 숫자와 아이콘 사이에서 제목 오른쪽(완료 버튼 왼쪽)으로 이동
   - 레이아웃 구조: 숫자 → 아이콘 → 제목 → 태그 → 완료 버튼
   - 태그가 한 줄에 표시되어 수직 공간 절약

3. **태그 색상 미묘한 회색 톤으로 변경**
   - 색상: `var(--text-muted)` (미묘한 회색)
   - `opacity: 0.7` 적용하여 덜 강조
   - `text-shadow`, `font-weight: bold` 제거
   - `font-weight: normal`로 변경

4. **Task 높이 줄이기**
   - Task 항목의 `padding`: `12px 0` → `8px 0`
   - 태그가 제목 아래에 있지 않아 수직 공간 절약

5. **아이콘과 숫자 간격 줄이기**
   - 아이콘의 `margin-right`: `12px` → `8px`
   - 아이콘의 `margin-left` 제거 (태그가 더 이상 아이콘 왼쪽에 없음)

6. **XP 표시 완전 제거**
   - XP 배지(`+{task.xp} XP`) 완전 제거
   - `.xp-badge` CSS 클래스 제거
   - 태그 옆에 표시되던 XP 관련 요소 모두 제거

7. **완료 버튼 크기 줄이기**
   - 완료 버튼(`.check-btn`) 크기: `40px × 40px` → `32px × 32px`
   - 체크 아이콘(`.check-icon`) 크기: `18px × 18px` → `16px × 16px`, `font-size: 1.2rem` → `1rem`
   - 원형 아이콘(`.circle`) 크기: `18px × 18px` → `16px × 16px`

8. **Focus 카테고리 아이콘 SVG로 변경**
   - 이모지(🎯) → SVG 타겟 아이콘으로 변경
   - `design.png`와 동일한 모노크롬 스타일 적용
