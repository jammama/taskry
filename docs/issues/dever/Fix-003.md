## 📖 기능 개요

### 목적
Tasks의 디자인을 초기 디자인(`design.png`)과 동일하게 변경합니다. 태그의 색상을 강하게 하지 않고, XP를 별도로 표시하지 않아 UI를 더 깔끔하고 미니멀하게 만듭니다.

### 사용자 스토리
- **As a** 사용자
- **I want** Tasks가 깔끔하고 미니멀한 디자인
- **So that** UI가 복잡하지 않고 집중할 수 있다

---

## 🎯 기능 요구사항

### 핵심 기능
1. Tasks 디자인을 `design.png`와 동일하게 변경
2. 태그 색상을 강하게 하지 않음 (미묘한 회색 톤)
3. XP를 별도로 표시하지 않음 (숫자 제거)
4. 카테고리 정보는 유지하되 시각적으로 덜 강조

### 상세 요구사항
- 태그는 미묘한 회색 톤으로 표시 (강한 색상 제거)
- XP 배지(`+{task.xp} XP`) 완전 제거
- Tasks는 깔끔하고 미니멀한 디자인 유지
- 카테고리 아이콘은 유지 가능 (선택)
- 초기 디자인(`design.png`)의 스타일을 참고하여 구현
