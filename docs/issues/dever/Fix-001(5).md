## 📖 기능 개요

### 목적
스와이프 중인 할 일에 구겨진 느낌의 시각적 효과를 추가합니다. 스와이프 정도에 비례하여 할 일이 구겨지는 효과를 구현하여 사용자에게 직관적인 피드백을 제공합니다.

### 사용자 스토리
- **As a** 사용자
- **I want** 스와이프할 때 할 일이 구겨지는 시각적 효과
- **So that** 스와이프 동작이 더 직관적이고 재미있다

---

## 🎯 기능 요구사항

### 핵심 기능
1. 스와이프 30px 이상일 때 구겨진 효과 적용
2. 스와이프 정도에 비례하여 구겨지는 정도 조절
3. 홀수/짝수 항목이 번갈아가며 다른 방향으로 구겨짐
4. 밝기 및 대비 조정으로 구겨진 느낌 강조

### 상세 요구사항
- **조건**: 스와이프 30px 이상일 때 `.crushed` 클래스 적용
- **구겨진 효과 계산**:
  - `scaleY`: 1 → 0.9 (최대 10% 축소, 스와이프 정도에 비례)
  - `scaleX`: 1 → 0.95 (최대 5% 축소, 스와이프 정도에 비례)
  - `skewX`: 스와이프 정도에 비례하여 최대 ±3deg
- **홀수/짝수 구분**: 홀수 항목은 오른쪽으로 기울고, 짝수 항목은 왼쪽으로 기울어짐
- **필터 효과**: `brightness(0.85) contrast(1.1)`
- **실시간 계산**: JavaScript로 스와이프 정도를 계산하여 style 속성에 직접 적용

## 개발 가이드

### 구현 순서
1. `.crushed` CSS 클래스 정의
2. 스와이프 정도 계산 로직 (JavaScript)
3. `style` 속성에 동적 transform 적용
4. 홀수/짝수 구분 로직 (index 기반)
5. 필터 효과 적용

### 주의사항
- 스와이프 정도는 `Math.abs(swipeCurrentX - swipeStartX)`로 계산
- 구겨진 효과는 스와이프 30px 이상일 때만 적용
- JavaScript로 계산된 값을 style 속성에 직접 적용 (CSS calc 함수 호환성 문제 방지)
- `will-change: transform` 추가하여 성능 최적화
