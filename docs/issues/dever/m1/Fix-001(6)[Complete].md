## 📖 기능 개요

### 목적
Task 내의 삭제 버튼(X 아이콘)을 제거하고, clip-path로 잘려나간 왼쪽 부분에 네온 글로우 효과를 추가합니다. 이를 통해 UI를 더 깔끔하게 만들고, 잘려나간 부분이 시각적으로 더 명확하게 보이도록 합니다.

### 사용자 스토리
- **As a** 사용자
- **I want** Task 내의 삭제 버튼이 없고, 잘려나간 부분에 네온 글로우 효과가 있는 것
- **So that** UI가 더 깔끔하고 잘려나간 부분이 더 명확하게 보인다

---

## 🎯 기능 요구사항

### 핵심 기능
1. Task 내의 삭제 버튼(X 아이콘) 완전 제거
2. clip-path로 잘려나간 왼쪽 부분에 네온 글로우 효과 추가
3. 스와이프 중이거나 선택된 Task의 잘려나간 부분에 글로우 효과 표시

### 상세 요구사항
- **삭제 버튼 제거**: 선택된 Task의 오른쪽에 표시되던 X 아이콘 완전 제거
- **네온 글로우 효과**:
  - clip-path로 잘려나간 왼쪽 가장자리에 네온 글로우 효과 추가
  - 잘려나간 정도에 비례하여 글로우 효과 적용
  - 핑크/빨간색 네온 글로우 (삭제를 암시하는 색상)
- **시각적 효과**: 잘려나간 부분이 마치 네온 라이트로 표시된 것처럼 보임

## 작업 완료 내역

### 변경 사항
- `src/lib/components/PlanningMode.svelte`: 삭제 버튼 제거 및 잘려나간 부분 네온 글로우 효과 추가

### 주요 수정 사항
1. **삭제 버튼(X 아이콘) 제거**
   - 선택된 Task의 오른쪽에 있던 `delete-indicator` 요소 완전 제거
   - SVG X 아이콘 제거
   - UI가 더 깔끔해짐

2. **스와이프 중 clip-path 적용**
   - 스와이프 중인 항목에도 clip-path 적용
   - 왼쪽으로 스와이프한 경우에만 clip-path 적용
   - 스와이프 정도에 비례하여 왼쪽 부분이 잘림 (최대 100px)

3. **잘려나간 왼쪽 부분에 네온 글로우 효과 추가**
   - `li.clipped` 클래스 추가 (스와이프 중이거나 선택된 Task)
   - `::before` pseudo-element로 왼쪽 가장자리에 글로우 효과 구현
   - 핑크/빨간색 네온 글로우 (#ff6b9d)
   - `linear-gradient`로 부드러운 글로우 효과
   - 다중 레이어 `box-shadow`로 깊이감 있는 글로우
   - `glow-pulse` 애니메이션으로 펄스 효과 (2초 주기)

4. **시각적 효과**
   - 잘려나간 부분이 마치 네온 라이트로 표시된 것처럼 보임
   - 스와이프 중 실시간으로 글로우 효과 업데이트
   - 선택 상태에서도 글로우 효과 유지

### 테스트 결과

- ✅ 스와이프 중 잘려나간 부분에 네온 글로우 효과 확인
- ✅ 선택된 Task의 잘려나간 부분에 네온 글로우 효과 확인
- ✅ 네온 글로우 펄스 애니메이션 확인

  - `@keyframes glow-pulse` 애니메이션
