# 이슈 프롬프트 템플릿

이 파일은 이슈 작성 시 참고할 수 있는 템플릿입니다.

**중요**: 모든 이슈는 `docs/issues/[할당자폴더]/[이슈타입]-[번호].md` 형식으로 작성해야 합니다.

## 할당자 폴더
- `dever/`: 개발자 (Developer)
- `planner/`: 기획자 (Planner)  
- `designer/`: 디자이너 (Designer)

## 이슈 파일 예시

`docs/issues/dever/Bug-001.md` 파일을 참고하세요.

## 템플릿 구조

```markdown
**[Assignee]:** @AI_[할당자명]

**[Issue]:** #[이슈타입]-[번호] (이슈 설명)

**[Requirement]:** 
[요구사항 상세 설명]

**[Details]:**
[상세 내용]

**[Development Order]:**
[개발 순서]

**[Target Files]:** 
[대상 파일 목록]

**[Dependencies]:** 
[의존성]

**[Guideline Check]:** 
[가이드라인 체크리스트]
```
